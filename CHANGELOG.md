# 변경 내역

## v0.4.6

### 새 기능

- **장면 이미지 업로드** — 각 장면에 참고 이미지를 첨부하여 시각적으로 확인할 수 있습니다.
- **에피소드별 장면 접기** — 장면 섹션에서 에피소드를 접고 펼쳐 원하는 구간만 볼 수 있습니다.

### 개선

- 검색바 가시성 개선 — 채팅 검색 시 보라색 강조로 눈에 잘 띕니다.

---

## v0.4.5

### 새 기능

- **채팅 이미지 첨부** — 📎 버튼으로 이미지를 첨부하면 AI가 이미지를 분석하여 답변합니다.
- **이미지 영구 저장** — 첨부한 이미지가 채팅 기록에 보존되어 언제든 다시 확인할 수 있습니다.

---

## v0.4.4

### 새 기능

- **채팅 내 검색** — 대화 중 특정 내용을 빠르게 찾을 수 있습니다. 검색창에서 ▲▼ 키로 이전/다음 결과 이동, 일치 항목이 하이라이트됩니다.
- **문장 스크랩** — 대화 중 마음에 드는 문장을 드래그하면 "스크랩" 버튼이 나타납니다. 저장된 스크랩은 프로젝트 좌측 패널에서 확인할 수 있습니다.
- **스마트 스크롤** — AI가 답변을 작성하는 중에도 위로 스크롤하여 이전 내용을 자유롭게 읽을 수 있습니다. 하단으로 돌아가려면 ↓ 버튼을 누르세요.
- **입력창 자동 확장** — 긴 내용을 입력할 때 입력창이 최대 3줄까지 자동으로 늘어납니다.

### 개선

- 삭제 확인 시 접근성 개선 (키보드 ESC 닫기, 포커스 트랩)
- 오류 발생 시 사용자에게 토스트 메시지로 안내
- API 요청 시 30초 타임아웃 적용으로 무한 대기 방지
